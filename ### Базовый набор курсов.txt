### Базовый набор курсов для создания статичных сайтов

#### 1. **Основы HTML и CSS**
* **Курс «HTML и CSS с нуля»**
  * Изучение базовой разметки
  * Работа с семантической версткой
  * Основы CSS-стилей
  * Позиционирование элементов
  * Создание адаптивных макетов

#### 2. **Продвинутые техники CSS**
* **Курс по Flexbox и Grid**
  * Современные методы верстки
  * Создание сложных макетов
  * Оптимизация кода

* **Курс по CSS-анимациям**
  * Создание плавных переходов
  * Работа с эффектами
  * Оптимизация производительности

#### 3. **JavaScript для фронтенда**
* **Базовый курс JavaScript**
  * Основы языка
  * Работа с DOM
  * Создание интерактивных элементов

* **Курс по jQuery**
  * Упрощение работы с JavaScript
  * Создание анимаций
  * Обработка событий

#### 4. **Инструменты разработки**
* **Курс по Git и GitHub**
  * Управление версиями
  * Работа с репозиториями
  * Командная разработка

* **Курс по редакторам кода**
  * VS Code
  * WebStorm
  * Настройка окружения

#### 5. **Оптимизация и производительность**
* **Курс по оптимизации сайтов**
  * Минификация кода
  * Оптимизация изображений
  * Кэширование
  * SEO-оптимизация

#### 6. **Практические навыки**
* **Курс по прототипированию**
  * Создание макетов
  * Работа с Figma/Sketch
  * Проектирование интерфейсов

* **Курс по адаптивной верстке**
  * Медиа-запросы
  * Отзывчивый дизайн
  * Тестирование на разных устройствах

#### Рекомендуемые платформы для обучения:
* **Udemy** - практические курсы с реальными проектами
* **Coursera** - академические курсы от университетов
* **Stepik** - русскоязычные курсы с поддержкой
* **HTML Academy** - специализированные курсы по веб-разработке
* **GeekBrains** - комплексные программы обучения

#### Порядок изучения:
1. **Основы HTML и CSS** → 2 месяца
2. **JavaScript** → 3 месяца
3. **Инструменты разработки** → 1 месяц
4. **Продвинутые техники CSS** → 2 месяца
5. **Практические навыки** → 3 месяца

#### Дополнительные рекомендации:
* **Практика на реальных проектах**
* **Изучение документации**
* **Участие в сообществах разработчиков**
* **Создание портфолио**

После прохождения этих курсов вы сможете:
* Создавать адаптивные сайты
* Работать с современными технологиями
* Оптимизировать производительность
* Использовать системы контроля версий
* Создавать интерактивные элементы
